Features

Interstakt contains an interface for composing a Score, as well as an integrated controller that can map into the contents of a Score.

The Score has a hierarchical structure, where levels of scenic objects build up to create the overall Score.

The playout of a Score’s structure is determined by Interstakt’s trigger and parameter system, which allows for scenes to trigger each other and send parameter between each other.

In order to create these interactions, a user would utilize Interstakt’s mapping system. This holds a database for all of the elements within a Score, and allows for a user to connect elements together.

In addition to the Score, Interstakt also has integrated controllers, which can map into the compositional hierarchy and the timbral parameters of each voice. Each control has the ability to map to multiple destinations and translate its output differently to each destination.

Each control can also update its different mappings (control scenes) depending on what events occur in the score.

The system is fully interactive: a performer can control the composition using the controllers, and the composition’s playout can determine what parameters the controllers are mapped to at a given time.
